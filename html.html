The HTML Skeleton is the basic structure of every HTML document.

Two sided tags have an open and close tag. The close tag has a / before the element. You can also code tags inside other tags, as seen above in the codepen example. the "em" tag is inside the "p" tag

One sided tags don't have a closing tag. You can include the backslash at the end of the tag. In the example above, it shows the "br/" tag being used.

See the Pen example b by jpxfrdd (@jpxfrdd) on CodePen.

The code above shows attributes. Attributes provide more information about a tag. They have the attribute name followed by an equal sign and a value in quotes. Some attributes are the "img src" tag and the "id" tag.

The "img src" tag is able to add an image to the webpage. The alt bit at the end of the "img src" tag is a description of the image. The "id" tag can be used to select one specific bit of code for things like CSS.
